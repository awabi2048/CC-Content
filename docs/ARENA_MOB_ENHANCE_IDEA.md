- ゾンビ
  - 飛びかかり攻撃
  - 盾を使える
  - 弓を使える
- スケルトン
  - 速い矢を射る・射るのが速い
  - 曲射（直線軌道でなくやや上向きで分かりづらく射る）
  - 効果付きの矢を射る（火矢・ポーション矢）
  - 盾を使える
  - 骨ブーメラン的な
- ブレイズ
  - 3連射のところを5連射
  - たまに大きめの弾（ガストが使うやつ）
  - 浮いている状態で突進して範囲攻撃
  - 周囲を燃やす
  - 反撃で無条件で燃やす
  - 通常攻撃でノックバック
  - 倒されたときに爆発
- クモ
  - たまに透明な状態でスポーン
  - 糸を吐いて遠隔攻撃（洞窟クモなら毒など）
  - 大きく飛びかかり攻撃
  - 登ってるところから飛びかかり攻撃
  - 攻撃時に速度低下・採掘速度低下（攻撃速度も下がる）など
  - 飛び道具無効
- スライム
  - 大きく飛びかかり攻撃
  - 粘液を飛ばして遠隔攻撃、当たると移動速度・採掘速度低下
  - 周りのスライムと合体して巨大化
  - （強化じゃないけど）分裂は無駄に時間掛かるだけなので無効化
  - 自然回復
  - ノックバック強化
  - ポーションエフェクトを吸収したスライム：攻撃時にそのエフェクト
  - 飛び道具無効（柔らかすぎて刺さらないというイメージ）
- エンダーマン
  - （変な位置に行くと困るので）バニラのテレポートは無効化
  - 飛び道具は当たるのと当たらないのと
  - 周囲に盲目付与
  - 炎無効
  - テレポート発生時、一時的に消えて周囲に射線を伸ばし、その後ぶつかるかランダムな方向などにテレポートしてどこに行くか分かりづらく（射線に当たるとそこにテレポート＋ダメージ判定）
  - 飛び道具は当たらない場合は近くにテレポート
  - 攻撃時にプレイヤーのHP吸って回復
  - プレイヤーの攻撃時にたまに位置入れ替え
- ガーディアン
  - ビーム攻撃が素早い
  - 周囲のプレイヤーを水底に落とす
  - 弓無効
  - 急加速
  - 近づいたらダメージ（棘の鎧的な）
  - ビーム攻撃の間に針を飛ばして攻撃
  - プレイヤーから積極的に逃げる
- ゴーレム（ゾンビで出現）
  - ノックバック強化
  - 飛び道具無効（スライムとは逆に硬いため）
  - 炎無効
  - 地鳴らし：一定範囲にダメージ＋ノックバック
  - 磁化：鉄・ネザライト装備来てるとたまに引き寄せ

## 未実装分の要確認（実装前に決める）
- スケルトン: 速射/高速弾/曲射 -> 同時適用するか、排他にするか
- スケルトン: 効果付き矢 -> 効果種、付与確率、継続時間
- スケルトン: 盾使用 -> 盾ダウン閾値、ダウン時間、再ダウン可否
- ブレイズ: 5連射 -> 連射間隔、連射中の移動可否
- ブレイズ: ガスト弾 -> 発射頻度、爆発破壊の有無
- ブレイズ: 突進範囲攻撃 -> 判定半径、ノックバック量
- クモ: 透明スポーン -> 出現率、解除条件
- クモ: 糸遠隔 -> 射程、付与デバフ、持続時間
- クモ: 飛び道具無効 -> 全投射物対象か、矢のみか
- スライム: 合体巨大化 -> 合体条件、サイズ上限
- スライム: 分裂無効 -> 例外条件の有無（ボス個体など）
- スライム: 粘液遠隔 -> デバフ種類、重複可否
- エンダーマン: バニラTP無効化 -> 代替TPの頻度と距離
- エンダーマン: 攻撃時吸収 -> 吸収量、回復上限
- ガーディアン: 水底落とし -> 発動条件、拘束時間
- ガーディアン: 弓無効 -> 完全無効か、軽減か
- ゴーレム: 地鳴らし -> CD、範囲、壁越し判定
- ゴーレム: 磁化 -> 対象装備、引き寄せ強度、CD
